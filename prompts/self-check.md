# Self-Check Prompt (Haiku)

SELF-CHECK: Review the changes you just made against the acceptance criteria.

## Checklist
1. Does the code meet ALL acceptance criteria?
2. Does the build pass?
3. Are there any obvious bugs, missing null checks, or unhandled errors?
4. Did you accidentally modify any forbidden files?
5. Are all new dependencies pinned to exact versions?
6. Is there any PII being logged?

If any check fails, fix the issue now.
If all checks pass, do nothing.
